📘 프로젝트 소개
SSULREBAL.COM은 SEO 최적화를 위한 웹사이트로, Next.js와 Tailwind CSS를 활용하여 구축되었습니다. 이 프로젝트는 검색 엔진 최적화(SEO)를 고려한 웹사이트 개발을 목표로 하며, 사용자 친화적인 UI와 빠른 로딩 속도를 제공합니다.
